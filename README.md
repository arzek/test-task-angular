# Test Task Angular

Install
- install Angular CLI - `npm install -g @angular/cli` 
- clone app
- `yarn or npm install`
- `node server.js`

Run test
- `ng lint`
- `ng test`
- `ng e2e --port=5000`

Run app
- `ng serve`
-  open http://localhost:4200/

Task time:

- HTML - 4h
- FE - 3h
- Testing - 1.5h
- CI - 1h
